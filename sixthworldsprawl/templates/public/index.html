A website built for use with the TableTop Role-Playing Game, Shadowrun (3e)
Contained within, will be a host of generators and rollers to help play the game, along with other features such as digital character sheets and character management tools.
Plus features for anyone to enjoy, even non-players, such as overviews of featured characters and quotes from some of our funniest moments.

That's honestly the best way that I can put it.
For those familiar with that site, a lot of the features that I want to add to this site will be the same types of tools that you find there.
However, it is important to note They have a team, I am doing this by myself.

Only 3rd Edition is supported at this time.
This is very much a personal project and I am working on it because 3rd Edition is the version that I mostly play and the one that I want to work on first.
While some planned features like the generators (magical groups, corps, missions, etc) might be able to be used with other editions with minor tweaks they will be primarily made with 3rd edition rules in mind.
However, when the 3rd edition features are complete and I feel that it is in a good spot, I would very much like to start making the site work with all editions of the game.
